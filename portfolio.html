Semicolon

Created with discord.py, Semicolon is my longstanding passion project. Beginning as a random hodgepodge of commands, this bot quickly evolved into a powerful moderation bot used by several large servers. This bot utilizes a SQL database to store per-server configurations, numerous third-party APIs such as Perspective to assist with real-time moderation, and features a robust localization system which allows the bot to be used by users who don't speak English.

Live GDQ Schedule

Viewing the schedule of upcoming speedruns during a Games Done Quick event without opening the Twitch stream is considerably tedious which is why I began this Discord bot in 2019 to read schedule data from the GDQ API and mirror it to a Discord text channel. Alongside the full event schedule, the bot displays a short roster of upcoming runs as well as timestamped links to the start of past speedruns using data fetched from contributors on Reddit.

HTSea

HTSea is an April Fool's Day project created by the staff of HTwins STEM which pokes fun at cryptocurrencies and NFTs. Users were able to earn Diamonds, our "cryptocurrency," by mining real diamond ore on our Minecraft server or by solving Wordles in their web browser.

As the project lead, I outlined how exactly the event would work and what all was necessary for our goals to be reached. I helped direct team members and provided my assistance with fixing problems in the code. I also contributed a lot of code myself and solely developed the Minecraft plugin used to convert diamonds (the item) to Diamonds (the "cryptocurrency").

Yahoo! Groups Viewer

As described by Wikipedia, "Yahoo! Groups was a free-to-use system of electronic mailing lists offered by Yahoo!." While archives of this defunct website exist, they are notoriously hard to peruse, effectively resulting in messages from these old boards being lost to time. This presented an issue to speedrunning communities, particularly the Mario Kart community, who often shared their personal best times to these boards. To help resolve this problem, I developed this website to provide an archive of select boards from Yahoo! Groups which can all be easily read and searched. The site is backed by a Kotlin (Ktor) backend which serves data from files pre-processed by a Python script for rendering by the JavaScript (SvelteKit) web browser frontend.
